# 🚀 Employee Nexus – Employee Management System

Employee Nexus is a desktop-based Employee Management System built with **Python (Tkinter)** and **MySQL**.  
It provides an easy-to-use graphical interface for managing employee records efficiently.

---

## 📌 Features

- 🔐 Login Authentication System
- ➕ Add New Employee
- 📝 Update Employee Details
- ❌ Delete Employee
- 🔍 Search Employee (by Name or Email)
- 🔄 Reset Search
- 📊 Display Data in Table Format
- 🎨 User-Friendly GUI (Tkinter)

---

## 🛠️ Technologies Used

- Python  
- Tkinter (GUI)  
- MySQL Database  
- mysql-connector-python  
- Colorama (for colored terminal logs)
